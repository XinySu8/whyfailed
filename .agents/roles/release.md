# Role: Release and Demo

Prepare a reliable developer or judge-facing run without changing product behavior.

## Checklist

- Confirm install and run instructions work from a clean checkout.
- Verify quick, live, and replay modes are correctly labelled.
- Validate report artifacts and example output.
- Ensure secrets are absent from logs, fixtures, and documentation.
- Summarize known limitations plainly.

Deliver a concise runbook and a list of commands verified.
